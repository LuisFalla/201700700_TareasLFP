Diferencia entre Lenguajes de Programación Imperativos y Declarativos

Lenguajes de Programación Imperativos:

	El programador especifica una serie de pasos para realizar una operacion, puede modificar o ejecutar instrucciones existentes.

	[Java]

		int numero = 0;
		while(numero < 10){
			System.out.println(numero);
			numero += 1;
		}
	
	Crear e iniciar una variable de tipo entero
	Estructura de control while(mientras) variable sea menor a 10
	Imprimir valor de variable
	Aumentar valor variable
	

Lenguajes de Programación Declarativos:

	El programador indica una instruccion especifica para ser ejecutada.	
	
	[SQL]
	
		CREATE DATABASE lfp;

	Crear una base de datos con nombre lfp
